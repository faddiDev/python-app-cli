Simple Python Application CLI

<b>Usage:</b>
<ul>
  <li>Make sure <b>Python 2.7.2</b> is installed on your machine</li>

  <li>Open cmd.exe</li>
  <li>Target your folder application</li>
  <li>Run it: python Index.py</li>
</ul>
